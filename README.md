# AquaCar

Proyecto de título orientado a la gestión de reservas, servicios y clientes para un lavado automotriz.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- Node.js
- Express
- MySQL

## Estructura del proyecto

/frontend → interfaz del sistema  
/backend → servidor y API  
/database → script SQL y modelo de base de datos

## Funcionalidades principales

- Registro de usuarios
- Inicio de sesión por roles
- Registro de vehículos
- Reserva de servicios
- Panel administrador
- Gestión de servicios
- Cambio de estado de reservas

## Instalación

1. Importar base de datos desde carpeta database
2. Configurar variables en archivo .env
3. Ejecutar backend

npm install
npm start
